# Smart City Traffic Pattern Analysis

## Description
This project analyzes smart city traffic data using Python.
It identifies traffic trends, junction-wise congestion, and
time-based patterns for forecasting applications.

## Dataset
- Source: Smart City Traffic Dataset
- Records: 48,120
- Features: DateTime, Junction, Vehicles

## Technologies Used
- Python
- Pandas
- Matplotlib
- NumPy

## How to Run
```bash
pip install -r requirements.txt
python analysis.py
